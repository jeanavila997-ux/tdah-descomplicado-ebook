# Páginas de Vendas

Este diretório centraliza o código fonte das landing pages (Páginas de Vendas) associadas ao aplicativo do E-book PWA.
Ao centralizar as páginas aqui, facilitamos o controle de versão, manutenibilidade e compartilhamento de assets (logos, identidades visuais).

## Estrutura Atual

- `/shared`: Recursos globais (CSS, scripts de tracking, imagens comuns) usados em múltiplas páginas.
- `/tdah-descomplicado`: Página de vendas principal (Kit de Produtividade Neuro-Compatível).
- `/ansiedade-sob-controle`: (Futuro) Página de vendas para o segundo e-book.

## Como Usar
Coloque os arquivos exportados/hospedados (como os da Hostinger) em suas respectivas pastas. 
Mantenha os estilos globais e variáveis de marca sincronizados com o E-book interativo conforme o `PLANO_ALINHAMENTO_PAGINA_VENDAS.md`.
